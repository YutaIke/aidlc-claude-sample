# ユーザーストーリー生成計画

## 計画概要

| 項目 | 決定内容 |
|------|----------|
| **分類アプローチ** | ペルソナベース（ユーザーロールごとに整理） |
| **ストーリー粒度** | 中程度（画面や操作単位） |
| **受け入れ基準形式** | チェックリスト＋Given-When-Then（複雑な条件） |
| **優先順位付け** | MoSCoW法（Must/Should/Could/Won't） |
| **追加情報** | 基本情報＋技術的注意点 |

---

## ペルソナ定義

### 対象ペルソナ
1. **一般従業員（Employee）** - アイデア提出者
2. **評価パネルメンバー（Panel Member）** - アイデア評価者
3. **管理者（Administrator）** - システム管理者
