# ペルソナ定義

## 概要

Ideation Portalには3つの主要なペルソナが存在します。

| ペルソナ | 役割 | 主な目的 |
|---------|------|----------|
| 一般従業員 | アイデア提出者 | イノベーティブなアイデアを提案し、評価を受ける |
| 評価パネルメンバー | アイデア評価者 | 提出されたアイデアを公正に評価する |
| 管理者 | システム管理者 | システム全体を管理し、表彰を実行する |

---

## ペルソナ1: 一般従業員（Employee）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 田中 太郎（仮名） |
| **役職** | 一般社員 |
| **年齢層** | 25〜45歳 |
| **ITスキル** | 基本的なWebアプリ利用経験あり |

### 目的とモチベーション
- 業務改善や新規事業のアイデアを会社に提案したい
- 自分のアイデアが認められ、評価されることを望む
- 他の従業員のアイデアから刺激を受けたい

### 主な行動
1. アイデアを思いついたら下書きとして保存
2. 内容を推敲してから正式に提出
3. 自分のアイデアの評価結果を確認
4. 他の従業員のアイデアを閲覧

### フラストレーション
- 入力が複雑だと途中で諦める
- 評価基準が不透明だと不信感を持つ
- フィードバックがないと改善できない

### 成功の定義
- スムーズにアイデアを提出できる
- 評価結果とフィードバックを受け取れる
- 上位にランクインして表彰される

---

## ペルソナ2: 評価パネルメンバー（Panel Member）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 鈴木 花子（仮名） |
| **役職** | 部門マネージャー / シニアスタッフ |
| **年齢層** | 35〜55歳 |
| **ITスキル** | 業務システム利用経験豊富 |

### 目的とモチベーション
- 組織のイノベーションに貢献したい
- 公正で客観的な評価を行いたい
- 有望なアイデアを見つけて育てたい

### 主な行動
1. 割り当てられたアイデアを確認
2. 3つの基準（実現可能性、インパクト、革新性）でスコアリング
3. 建設的なフィードバックを記入
4. 評価状況を管理

### フラストレーション
- 評価すべきアイデアが多すぎると負担になる
- 評価基準が曖昧だと判断に迷う
- 他の評価者のスコアに影響されたくない

### 成功の定義
- 効率的に評価作業を完了できる
- 自分の評価が組織の意思決定に貢献する
- 有望なアイデアを正しく見極められる

---

## ペルソナ3: 管理者（Administrator）

### 基本情報
| 項目 | 内容 |
|------|------|
| **名前** | 佐藤 一郎（仮名） |
| **役職** | 人事部 / 経営企画部スタッフ |
| **年齢層** | 30〜50歳 |
| **ITスキル** | システム管理経験あり |

### 目的とモチベーション
- イノベーション文化を組織に定着させたい
- 公正で透明な評価プロセスを維持したい
- 優れたアイデアを適切に表彰したい

### 主な行動
1. ユーザーアカウントの管理（作成・編集・削除）
2. 評価パネルメンバーの任命・解除
3. ダッシュボードの更新
4. 上位アイデアの表彰実行

### フラストレーション
- ユーザー管理が煩雑だと時間がかかる
- 評価状況の把握が難しいと進捗管理できない
- 表彰プロセスが複雑だとミスが発生する

### 成功の定義
- 効率的にシステムを管理できる
- 評価プロセスが円滑に進行する
- 表彰が適切に実行され、従業員のモチベーションが向上する

---

## ペルソナ間の関係

```
+-------------------+
|    管理者         |
| (システム管理)    |
+-------------------+
        |
        | ユーザー管理・パネル任命
        v
+-------------------+     評価・フィードバック     +-------------------+
| 評価パネル        | <-------------------------> |   一般従業員      |
| メンバー          |                             | (アイデア提出者)  |
+-------------------+                             +-------------------+
        |                                                   |
        | スコアリング                                      | アイデア提出
        v                                                   v
+---------------------------------------------------------------+
|                      Ideation Portal                          |
|                   (アイデア・評価データ)                       |
+---------------------------------------------------------------+
```
